# Pyanodons Quick Start

## Overview

Pyanodons Quick Start gives power armor MK2 and pY construction robot MK4 to help players start Pyanodons saves faster.

## Features

- Gives each eligible player 100 `py-construction-robot-mk04`.
- Gives each eligible player 1 `power-armor-mk2`.
- Inserts starter equipment into the armor grid, including night vision, belt immunity, solar panels, exoskeletons, personal roboports, and quantum batteries.
- Records which players have already received the package so the same player does not receive it repeatedly.
- Recommends helper mods that fit a Pyanodons quick-start save: Dynamic Inventory for larger inventory space, Kux Personal Teleport for teleportation, Far Reach for longer interaction reach, and Section Autocraft for faster automated hand crafting.

## Usage

- Start or join a Pyanodons save with a valid character, and the mod will grant the package automatically.
- The grant check runs on mod initialization, when a player joins the game, and when a cutscene finishes or is cancelled.
- The mod requires `base` and `pyalternativeenergy`.
- Recommended optional mods are `DynamicInventory`, `Kux-PersonalTeleport`, `far-reach`, and `section-autocraft`. They are not required for the starter package, but they make long Pyanodons starts easier to manage.
- If a player has no valid character yet, the mod waits for a later eligible event instead of granting the package immediately.

### Example

When a new player joins a Pyanodons save with a valid character, the mod gives 100 `py-construction-robot-mk04`, 1 `power-armor-mk2`, and the listed equipment in the armor grid once for that player.

# 中文说明

## 概览

Pyanodons 快速开局会给予能量装甲 MK2 和 pY 建设机器人 MK4，帮助玩家更快开始 Pyanodons 存档。

## 功能简介

- 为符合条件的玩家发放 100 个 `py-construction-robot-mk04`。
- 为符合条件的玩家发放 1 件 `power-armor-mk2`。
- 自动向装甲网格中放入夜视、锚定、太阳能模块、机械外骨骼、机器人指令模块和量子电池等开局装备。
- 记录已经领取过装备包的玩家，避免同一玩家重复领取。
- 推荐搭配适合 Pyanodons 开局的辅助模组：Dynamic Inventory 提供大背包，Kux Personal Teleport 提供传送，Far Reach 提供远触，Section Autocraft 提供更快的自动手搓。

## 使用说明

- 在拥有有效角色的情况下开始或加入 Pyanodons 存档，模组会自动发放装备包。
- 模组会在初始化、玩家加入游戏、过场动画结束或取消时检查是否需要发放装备。
- 模组必需依赖 `base` 和 `pyalternativeenergy`。
- 推荐的可选模组是 `DynamicInventory`、`Kux-PersonalTeleport`、`far-reach` 和 `section-autocraft`。它们不是领取开局装备的必要条件，但能明显改善漫长的 Pyanodons 开局体验。
- 如果玩家暂时没有有效角色，模组会等待后续满足条件的事件，而不是立刻发放装备。

### 例子

当新玩家带着有效角色进入 Pyanodons 存档时，模组会为该玩家发放 100 个 `py-construction-robot-mk04`、1 件 `power-armor-mk2`，并把列出的装备放入装甲网格；同一玩家只会领取一次。
